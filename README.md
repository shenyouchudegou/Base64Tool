# Base64Tool

一个简洁的图形化 Base64 编码与解码工具。

## 功能特点

-   **直观的图形界面**：基于 WinForms 开发，操作简单，上手即用。
-   **核心功能**：支持文本到 Base64 的编码以及 Base64 到文本的解码。
-   **快速转换**：输入即时转换，提高工作效率。
-   **结果交换**：提供一键交换输入框与输出框内容的功能，方便连续处理。

## 技术栈

-   **语言**：C#
-   **框架**：.NET 8.0 / .NET Framework 4.8
-   **UI 框架**：Windows Forms (WinForms)
